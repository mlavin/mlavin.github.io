Mark Lavin

Python Web Application Developer

I'm Technical Director at Caktus Consulting Group in Carborro, NC. I maintain a handful of open source projects mostly related to the Django framework. I'm currently working on a Python/Django book with Julia Elman for O'Reilly Media titled "Lightweight Django".

I'm passionate about beer and enjoy homebrewing when I can find the time. Along with Michael Manfre, I built and run Brewed By Us, a homebrewing recipe collection website. Most of the beers I've made can be found there.

When I'm not programming I'm usually out running or spending time with my wife and daughter.
